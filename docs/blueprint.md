# **App Name**: Cascade Rewards

## Core Features:

- Task Management: Display a list of available tasks, such as watching videos or reading content, and award points upon completion.
- Point Tracking: Track user points earned from completed tasks and referrals.
- Referral System: Generate a unique referral code for each user. When a new user joins using the referral code, the referrer earns points. Keep track of a referral network in a Cloud SQL database
- Referral Leveling: Calculate a bonus reward depending on what referral level a user has obtained
- User Dashboard: Display user profile info, current points balance, referral code, a list of users they have referred, and earnings from referrals.

## Style Guidelines:

- Primary color: Teal blue (#46A3FF), a calm and trustworthy color for the platform.
- Background color: Light gray (#F0F4F8), providing a clean and modern backdrop.
- Accent color: Soft lavender (#C39BFF), to highlight interactive elements and call to actions.
- Body and headline font: 'Inter' sans-serif for a clean, modern, neutral feel.
- Use a set of consistent, minimalist icons for tasks and navigation, focused on clarity and ease of recognition.
- Clean and intuitive layout with clear sections for tasks, points, and referrals to improve user experience.
- Subtle transitions and feedback animations to provide a smoother and more engaging experience when completing tasks.
